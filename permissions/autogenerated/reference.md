- `all-without-delete`

<td>

`android-fs:scope`

</td>
<td>

An empty permission you can use to modify the global scope.

## Example

```json
{
  "permissions": [
    "android-fs:all-without-delete",
    {
      "identifier": "android-fs:scope",
      "allow": [
        "$APPDATA/documents/**/*"
      ],
      "deny": [
        "$APPDATA/documents/secret.txt"
      ]
    }
  ]
}
```
